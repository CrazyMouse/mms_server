##已实现功能如下：
* 彩信接收及响应处理
* 彩信状态报告发送(已实现组包回复多条状态报告)状态报告异步发送
* http长链接支持
* 强制关闭状态报告
* 配置自动更新（修改mmsConfig.properties文件后10秒自动生效，周期可修改）
* 性能数据实时统计

## 性能测试
1. 测试环境：MAC OS 10.9，CPU:I7 4核8线程，内存：8G
2. 速度：80KB彩信包，平均140条每秒
